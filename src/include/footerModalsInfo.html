General

mix is an online store created with by Cristian Ilisei, which uses HTML5, CSS3, Font awesome, Bootstrap 5, Bootstrap notify, Javascript, Jquery, REST JSON API as technologies.The online store uses a "virtual" shopping cart where you can add products using the "add to cart" buttons. The shopping cart offers the possibility to change the quantities or delete products and automatically calculates total price.
